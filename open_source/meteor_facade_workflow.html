UV Read

  • Outputs (U, V, 0) per mesh vertex
UV Transform
Scale + Offset
  • VectorMath SCALE → UV Scale A / B
  • VectorMath ADD → UV Offset A / B
  • IMG_B gets +0.37, +0.61 decorrelation shift
GeometryNodeImageTexture × 2
Image Sampling
Sampled at transformed UV per vertex
Extension: REPEAT  ·  Interpolation: Linear
bit0 = Color_A > Threshold
bit1 = Color_B > Threshold
Bit Encoding
Zone Index 0 – 3
img_zone = bit0 + bit1 × 2
→ 0 / 1 / 2 / 3 (each ≈ 25 % surface)
Height Gradient
Gradient Mask
  • World position projected along rotatable axis
  • Normalised by Height Min / Max
  • MapRange SMOOTHSTEP → gradient value
  • TexNoise blur (Scale 12, Detail 2)
  • use_img = gr_n > 0.5
Zone Multiplier

eff_zone = img_zone × use_img

Per-face zone selection: |eff_zone − zi| < 0.5

GeometryNodeSeparateGeometry
Separate Zone Faces

  • Input: boundary-subdivided mesh
  • Output: only this zone's faces
GeometryNodeSubdivideMesh
Panel Grid Subdivision
  • Level = Grid Zi (0–5)
  • 4^N panels per original face
  • Independent per zone
Z0: 2  ·  Z1: 3  ·  Z2: 4  ·  Z3: 2 (defaults)
GeometryNodeScaleElements
Gap Between Panels
  • Domain: FACE
  • Scale = 1 − Gap Size
  • Shrinks each face → visible joint
ShaderNodeTexNoise + Math
Per-Panel Extrude Depth
  • TexNoise 3D (Detail 4, Roughness 0.7)
  • Scale = Height Noise
  • Fac × Depth Zi → per-face offset
depth = noise.Fac × Depth Zi
GeometryNodeExtrudeMesh
Extrude Panels
  • Mode: FACES
  • Individual: True
  • Offset Scale: depth (noise-driven)
  • Each face extruded independently
