Path A — Shape Detection
Step A1
Apply Threshold
  • Binary mask from normalized values
  • Pixel < threshold → active shape
  • Invert toggle reverses polarity
Step A2
Connected Components
  • DFS flood fill — 4-connected
  • Groups adjacent active pixels
  • Min Size filter removes noise
  • Up to 8 000 components rendered
Step A3
PCA Shape Analysis
  • Centroid (cx, cy)
  • Covariance matrix → eigenvalues λ₁ λ₂
  • Orientation angle from eigenvectors
  • OBB length × width (rotated bbox)
  • Circularity = 4π·area / perimeter²
  • Aspect ratio = λ₁ / λ₂
Step A4
Shape Classification
  • dot — circularity > 0.7
  • line — aspect ratio > 6
  • rect — aspect ratio > 2.5
  • spline — smooth convex hull
  • Gemini patternType biases selection
